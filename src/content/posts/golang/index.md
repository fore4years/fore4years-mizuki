# Golang语法
## 变量的定义与声明
### 单变量
变量的定义与声明不同于JAVA语法(`int a = ...`),具有4种定义方法:  
1. `a int`   ----> 只定义不赋值,后续赋值。
2. `a := ..`;
3. `var a int = ..`;
4. `var a = ...`;

### 多变量
如果想对多个变量批量定义或复制,可以采用以下几种方式:
* `var a,b,c int`
* `var(a int, b int, ...)`

:::important
`_ = ..`此方式为匿名赋值,`_`是匿名变量,向它赋值的值会被抛弃,无法使用。
:::

## `iota`枚举
对于有一定规则的常量声明,挨个声明过于麻烦,可以使用`iota`关键字进行声明  
使用`iota`关键字那一行将被赋予0,此后每一行`iota`递增1,例如:
```go
const (
    a = iota  ---> a = 0,a = iota
    b         ---> b = 1,b = iota + 1
    c         ---> c = 2,c = iota + 2
)
```
> `iota`关键字用于常量枚举,因此只能用于**常量**,也称**常量声明的计数器**

## 数组
对于数组的声明,语法如下:
```go
var a [n]int
var a [...]int{1, 2, 3, ...}自己推导数组长度
```
此时只是声明a为一个**指向int数组的指针**,该数组**还未为其分配存储空间**。  
如果在`[n]int`中指定多大的大小(即n),那么就会为其**开辟多大的空间,并赋予默认值**。  
* 对于`var a [n]int{1, 2, 3, ..., m}`这种赋值方式来说: n是指定开辟多大的空间,那么会有如下情况: 
1. `n > m`, 剩余空间由类型默认值填充
2. `n < m`, **如果超出空间部分,将会发生编译报错**
